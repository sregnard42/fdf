ESCAPE : Exit program

W : Move up
A : Move to the left
S : Move to the right
D : Move down

Q : Zoom
E : Unzoom

Up Arrow : Increase height
Down Arrow : Decrease height

Left Arrow : Go to previous view
Right Arrow : Go to next view

. : Reset height/zoom to default values
